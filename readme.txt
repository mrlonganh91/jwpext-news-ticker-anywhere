If you wish to display vertical scrolling news on your WordPress site, choose News Ticker Anywhere. 
It’s the simple and easy WordPress news ticker plugin that lets you manage any number of the post. 

== Description ==

News Ticker Anywhere lets you add and display news or your post categories with shortcodes. 
This fantastic WordPress news ticker plugin is compatible with custom post types. 
Its customizable shortcode can be adjusted to change the attribute or select specific post categories. 
The plugin provides you features like adding custom color, effects & animations, etc.

= Features =
* Simple & Intuitive
* Powerful Functions
* 3 Default Ticker Modes
* Assigning Items Post to Multiple Categories.
* Add Unlimited News.
* Responsive Layout Frontend.
* Thumbnail Image Support.
* Shortcode Support.
* Support effects & animations.
* Support Shortcode Widget.
* .pot file for translations included

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "News Ticker Anywhere"
3. **Install and Activate** News Ticker Anywhere from your Plugins page
4. **Click** on the new menu item "News Ticker Anywhere" and config the Plugin!
